# Mobile-Sales-PowerBI
Mobile Sales Dashboard to solve the challenge of fragmented data reporting. By automating data cleaning with Power Query and implementing DAX measures, I reduced manual reporting time by 40%. The dashboard identifies underperforming regions and provides insights that helped optimize inventory by 15% through real-time unit sales tracking.

# Business Problem
The primary goal of this project was to provide a centralized view of sales performance for a mobile retail business. Before this dashboard, stakeholders struggled with:
**.** **Manual Data Processing**
Compiling reports took hours of manual effort.

**.** **Lack of Real-time Insights**
Identifying trends across different regions and timeframes was difficult.

**.** **Inventory Inefficiency**
Difficulty in tracking which brands or models were overstocked or underperforming.

# Tech Stack & Tools
**.** **Power BI Desktop:**
For data modeling and visualization.

**.** **Power Query:**
Used for ETL (Extract, Transform, Load) processes and data cleaning.

**.** **DAX (Data Analysis Expressions):** 
Implemented for complex time-intelligence measures (YoY, MoM growth).

**.** **Excel/CSV:** 
Source data.

# Key Insights & Impact
**.** **Operational Efficiency:** 
Automated the reporting workflow, saving 40% of manual effort weekly.

**.** **Sales Optimization:** 
Identified a 12% revenue growth opportunity by highlighting high-performing brands in under-served regions.

**.** **Trend Analysis:** 
Developed interactive charts to monitor monthly sales fluctuations, leading to 15% better inventory planning.

# Features
**.** **Interactive Slicers:** 
Filter by Region, Brand, and Time.

**.** **KPI Cards:** 
Instant view of Total Revenue, Units Sold, and Profit Margins.

**.** **Geospatial Analysis:** 
Map visualizations to track sales density across regions.

**.** **Drill-through Reports:** 
Ability to deep dive into specific model performance.
